Тестовый проект

Язык: Swift.
Интерфейс: UIKit + AutoLayout.
Сеть: URLSession.
Локальное хранилище: CoreData.
Архитектура экрана: упрощённый MVP
(FeedPresenter + FeedViewController + StorageManager / NetworkManager)

Основные возможности
Загрузка постов с https://jsonplaceholder.typicode.com/posts
Сохранение постов в Core Data (сущность PostEntity)
Офлайн‑режим:
первый запуск онлайн — данные кешируются;
последующие запуски без интернета читают посты из Core Data.
Пагинация на клиенте:
все посты хранятся в allPosts;
таблица показывает по pageSize элементов, подгружая следующую «страницу» при прокрутке вниз.

Лайки:
отдельная сущность LikeEntity в Core Data;
лайки привязаны к postId и userId;
в UI отображается иконка лайка (♡ / ♥︎) и счётчик.

Бонусные задания:
центральный индикатор загрузки при первом старте;
плавная анимация появления таблицы;
карточный дизайн ячейки (скругления, «карта»);
анимация кнопки лайка (scale + spring);
алерты при ошибках сети/данных.
